1 Project Overview and Bookdown

This project advocates for the use of bookdown in rendering data science projects and executive reports unlike the traditional reporting standards in pdf format.Traditional format of rendering books or reports in Latex formats and Word documents can be quite challenging in regards to typsetting details and commands. However, what if there’s an efficient way to render books to readers. What if users want to leave comment for authors, what if users want to interact with plots in book, what if clients want to access report without storing pdf’s documentations on their personal drives.

Bookdown can help you prepare a book by rendering multiple (R Markedown) files into single book. In most instances users can interact with object in the book by changing certain parameters in a statistical model etc. The major focus of this project is to demonstrate how author’s and data scientist can bridge the gap between users and clients. Also, knowledge of HTML/CSS is not really needed in order to create a book. Check out a demo book by bookdown and books created from bookdown Examples.
